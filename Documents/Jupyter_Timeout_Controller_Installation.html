Jupyter Timeout Controller Installation

Note: To install the bookmark, drag the button below to your bookmarks bar. If you can't drag it, follow the manual installation steps below.

Option 1: One-Click Installation

Drag this button to your bookmarks bar:

Jupyter Timeout Control

Option 2: Manual Installation

1. Right-click on your browser's bookmark bar
2. Select "Add bookmark" or "New bookmark"
3. Enter "Jupyter Timeout Control" as the name
4. Copy and paste the following code as the URL:
javascript:(function(){const s=document.createElement('script');s.src='https://cdn.jsdelivr.net/gh/wanglrebe/Jupyter-alive@main/timeout-controller.min.js';document.body.appendChild(s)})();

3. 将名称设置为"Jupyter 超时控制"
4. 将以下代码复制粘贴为网址：
javascript:(function(){const s=document.createElement('script');s.src='https://cdn.jsdelivr.net/gh/wanglrebe/Jupyter-alive@main/timeout-controller.min.js';document.body.appendChild(s)})();
